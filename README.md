# toggle-quotes.nvim

### Description

Toggle between simple quotes and double quotes.

### Installation

With [packer.nvim](https://github.com/wbthomason/packer.nvim):

```lua
use 'paul-louyot/toggle-quotes.nvim'
```

### Usage

From the vim command line:
```
:ToggleQuotes
```
